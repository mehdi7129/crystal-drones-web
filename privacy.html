Politique de Confidentialité

Dernière mise à jour : 30 août 2025

1. Introduction

Crystal Drones SAV ("nous", "notre", "nos") s'engage à protéger votre vie privée. Cette politique de confidentialité explique comment nous collectons, utilisons, et protégeons vos informations lorsque vous utilisez notre application mobile.

⚠️ Application à usage interne uniquement

Cette application est exclusivement réservée aux employés et techniciens autorisés de Crystal Drones SAV. Elle n'est pas destinée au grand public et nécessite des identifiants professionnels fournis par l'entreprise pour y accéder. Aucune donnée client externe n'est collectée via cette application.

2. Informations que nous collectons

Nous collectons uniquement les informations des employés Crystal Drones :

  • Identifiants professionnels : Nom d'utilisateur et email professionnel fournis par l'entreprise
  • Données techniques internes : Informations sur les drones de l'entreprise, interventions SAV internes, rapports techniques
  • Documentation interne : Photos des équipements Crystal Drones pour le suivi SAV
  • Données opérationnelles : Localisation des interventions sur les sites Crystal Drones

3. Utilisation des informations

Les informations collectées sont utilisées exclusivement pour les opérations internes :

  • Gérer les accès des employés Crystal Drones à l'application
  • Suivre et documenter les interventions SAV internes
  • Optimiser les processus de maintenance internes
  • Communication interne entre les équipes techniques

4. Partage des informations

Les données restent strictement internes à Crystal Drones :

  • Accès limité aux employés Crystal Drones autorisés uniquement
  • Aucun partage avec des tiers ou clients externes
  • Données conservées sur des serveurs internes sécurisés
  • Conformité aux obligations légales si requises

5. Sécurité des données

Nous mettons en œuvre des mesures de sécurité avancées pour protéger vos informations :

Chiffrement et Certificats SSL/TLS

  • HTTPS obligatoire : Toutes les communications sont chiffrées via SSL/TLS
  • Validation des certificats : Vérification personnalisée des certificats SSL pour crystal-drones.duckdns.org
  • Certificate Pinning : Validation renforcée contre les attaques man-in-the-middle
  • App Transport Security (ATS) : Configuration iOS stricte interdisant les connexions non sécurisées

Authentification Multi-Niveaux

  • Tokens JWT sécurisés : Authentification par tokens avec expiration automatique
  • Biométrie (Face ID/Touch ID) : Authentification biométrique pour un accès rapide et sécurisé
  • iOS Keychain : Stockage chiffré des identifiants dans le trousseau sécurisé iOS
  • Refresh automatique : Renouvellement automatique des tokens expirés

Protection des Données Locales

  • Chiffrement iOS natif : Toutes les données locales sont chiffrées au repos par iOS
  • Core Data sécurisé : Base de données locale protégée par le système iOS
  • Keychain Services : Utilisation exclusive du trousseau iOS pour les données sensibles
  • CryptoKit Framework : Utilisation des API cryptographiques modernes d'Apple

Sécurité Réseau

  • Sessions éphémères : Pas de mise en cache des données sensibles
  • Timeout de connexion : Déconnexion automatique après inactivité
  • Détection de proxy : Validation des connexions réseau
  • Headers sécurisés : Utilisation de Bearer tokens dans les en-têtes HTTP

Sécurité de l'Application

  • Détection de jailbreak : Vérification de l'intégrité du dispositif
  • Validation de signature : Vérification de l'intégrité de l'application
  • SHA256 Hashing : Hachage sécurisé pour la vérification des mots de passe
  • Nettoyage sécurisé : Suppression complète des données lors de la déconnexion

6. Frameworks de Sécurité iOS

L'application utilise les frameworks de sécurité officiels d'Apple :

  • Security.framework : Gestion sécurisée du trousseau iOS
  • LocalAuthentication.framework : Intégration Face ID et Touch ID
  • CryptoKit.framework : Opérations cryptographiques modernes
  • Network.framework : Sécurité des connexions réseau

7. Stockage des données

Les données sont stockées de manière sécurisée :

  • Serveur Directus : Backend sécurisé avec authentification JWT
  • PostgreSQL : Base de données avec chiffrement des mots de passe (Argon2)
  • Docker : Conteneurisation isolée des services
  • Accès restreint : Seuls les employés autorisés peuvent accéder aux données

Conservation : Les données sont conservées pendant la durée d'emploi et peuvent être supprimées sur demande.

8. Vos droits

Conformément au RGPD, vous avez le droit de :

  • Accéder à vos données personnelles
  • Corriger vos données
  • Demander la suppression de vos données
  • Vous opposer au traitement de vos données
  • Demander la portabilité de vos données
